# 🚀 EasyAdmin

EasyAdmin est une interface d’administration simple, rapide et intuitive conçue pour faciliter la gestion et l’organisation de vos données et ressources.

---

## 📌 À propos

Ce projet a pour objectif de fournir une solution légère et facile à utiliser pour administrer des contenus, utilisateurs ou systèmes, sans complexité inutile.

---

## ✨ Fonctionnalités

- Interface simple et intuitive
- Gestion rapide des données
- Structure modulaire et facile à étendre
- Code propre et maintenable
- Installation rapide

## 📦 Installation

### 1. Cloner le repository

```bash
git clone https://github.com/Tablelkea/EasyAdmin.git
```

### 2. Aller dans le dossier

```bash
cd EasyAdmin
```
### 3. Installer les dépendances (si nécessaire)

```bash
npm install
```
### 4. Lancer le projet

```bash
npm start
```
